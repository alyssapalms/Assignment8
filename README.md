## Overview
This project applies a supervised machine learning algorithm (K-Nearest Neighbors) 
to classify Netflix titles as either Movies or TV Shows based on features such as 
release year, rating, duration, and genre.

## Dataset
- Source: [Netflix Trending Dataset](https://www.kaggle.com/datasets/shreyashpatil217/netflix-trending-dataset)

## Methods
- Algorithm: K-Nearest Neighbors (KNN)
- Evaluation Metrics: Accuracy, Confusion Matrix, Classification Report, ROC-AUC Curve

## Results
The model achieved good predictive accuracy and AUC scores, demonstrating its 
ability to distinguish between Movies and TV Shows.
